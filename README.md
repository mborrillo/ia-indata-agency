# 🚀 ia-indata-agency

**Tu template oficial de Agencia de Datos IA**  
Basado en el proyecto AgroTech Extremadura pero 100% replicable a cualquier sector (retail, salud, logística, etc.).

## Filosofía
"Tony Stark tiene la visión, Jarvis ejecuta con precisión."  
El humano (tú) es el único responsable en producción. La IA hace el 90% del trabajo.

## Cómo usar este template
1. Crea un nuevo `spec.md` para cada cliente.
2. Abre este repo en Cursor → la IA sigue automáticamente el pipeline SDD (Explorer → Proposer → Arquitecto → Verificador).
3. Todo entra por Pull Request con tests y seguridad automática.

## Stack Freemium (sin límites)
- Base de datos: **Neon.tech** (Postgres ilimitado)
- Apps: **Streamlit** (deploy gratis)
- Orquestación: **GitHub Actions** + Cursor
- Documentación: `/docs/` (AGENT.md, WORKFLOWS.md, skills, etc.)
